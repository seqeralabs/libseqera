# lib-data-broadcast-redis changelog

0.1.0 - 06 Apr 2026
- Initial release
- EventBroadcast<T> interface with pub/sub semantics and replay support
- LocalEventBroadcast<T> in-memory implementation for single-replica / dev / test
- RedisEventBroadcast<T> Redis Streams implementation (XADD/XRANGE) for multi-replica
- AbstractEventBroadcast<T> auto-selects local or Redis via nullable JedisPool injection
