# lib-lock-redis changelog

1.1.0
- Add watchdog auto-renewal that renews lock TTL at ttl/3 interval while holder is alive
- Use atomic Lua script for ownership-checked TTL renewal
- Add AtomicBoolean guard to prevent race between release and watchdog renewal
- Expose lib-lock as api dependency

1.0.0
- Initial release of lib-lock-redis module
- Implement RedisLockManager using JedisPool for distributed locking
- Implement RedisLock with Lua script for atomic release
- Use Redis SET NX PX for atomic lock acquisition
- Support configurable auto-expire duration and acquire retry interval
